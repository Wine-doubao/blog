    94 二叉树的中序遍历

    doubao 2022-3-25 LeetCode二叉树中序遍历

    标签：二叉树 中序遍历 简单

    # 题目

    给定一个二叉树的根节点 root ，返回 它的 中序 遍历 。

    示例 1：

    输入：root = [1,null,2,3]
输出：[1,3,2]
    1
    2

    示例 2：

    输入：root = []
输出：[]
    1
    2

    示例 3：

    输入：root = [1]
输出：[1]
    1
    2

    提示：

    • 树中节点数目在范围 [0, 100] 内
    • -100 <= Node.val <= 100

    进阶: 递归算法很简单，你可以通过迭代算法完成吗？

    # 解析

    好久没做二叉树的算法题了，有点懵，去看了下题解，然后尽量自己写一遍。一共有3种解法：

    1. 递归
    2. 迭代
      • 递归和迭代两种方式是等价的，区别在于递归隐式地维护了一个栈，而迭代需要显式地将栈模拟出来。
      • 两者的时间和空间复杂度都为O(n)。
    3. Morris遍历
      • Morris遍历算法能将非递归的中序遍历空间复杂度降为O(1)。不需要额外的栈空间来存储，而是通过找到节点在中序遍历中的前驱节点。前驱节点：按中序遍历访问树，如果访问结果为ABC，则称A为B的前驱节点，B为C的前驱节点。前驱节点就是遍历中排在其前面一个的节点。
      • 算法步骤如下（假设当前遍历到的节点为x）：
        1. 如果x无左孩子，则x加入结果，x=x.right
        2. 如果x有左孩子，找前驱节点pre（就是当前x向左走一步，然后一直向右走到终点）
          • pre右孩子为空，说明其左子树还没访问，则右孩子指向x，x=x.left
          • pre右孩子不为空，说明其左子树已经访问完了，则x加入结果，x=x.right，并将pre的右孩子置空
        3. 重复上述操作，直至访问完整棵树。
      • 优化的点：将 x 的左子树中最右边的节点的右孩子指向 x，这样在左子树遍历完成后我们通过这个指向走回了 x，且能通过这个指向知晓我们已经遍历完成了左子树，而不用再通过栈来维护，省去了栈的空间复杂度。
      • 时间复杂度：O(n)，空间复杂度：O(1)。其中 n 为二叉树的节点个数。Morris 遍历中每个节点会被访问两次，因此总时间复杂度为 O(2n)=O(n)。

    点击查看官方题解 (opens new window)。

    /**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number[]}
 */
var inorderTraversal = function(root) {
  let res = [];

  // 递归
  let inorder = (root) => {
    if(!root){
      return;
    }
    inorder(root.left);
    res.push(root.val);
    inorder(root.right);
  }
  inorder(root);

  // 循环
  let stack = [];// 用于存放遍历过的父节点
  while(root || stack.length){
    while(root){
      stack.push(root);
      root = root.left;
    }
    root = stack.pop();
    res.push(root.val);
    root = root.right;
  }

  // Morris中序遍历
  let pre; // 当前节点的前驱节点
  while(root){
    if(root.left){
      // 找到前驱节点
      pre = root.left;
      while(pre.right && pre.right!==root){
        pre = pre.right;
      }
      if(pre.right) {// 说明其左子树已经访问过了
        root = pre.right;
        res.push(root.val); 
        root = root.right; // 继续访问右子树
        pre.right = null; // 复原
      } else{// 刚找到前驱节点，改变指向后开始访问左子树
        pre.right = root;
        root = root.left;
      }
    } else{// 没有左子树，直接访问右子树
      res.push(root.val);
      root = root.right;
    }
  }

  return res;
};
